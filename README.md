# GTA Radio External
A tool that mutes/unmutes external music players depending on whether the radio would play in GTA.

Features:  
-Supports GTA3, VC & SA  
-Supports Winamp v5.666 and Foobar2000 v1.3.14  
-Can Mute or Pause music when radio stops
